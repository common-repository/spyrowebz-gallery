== Description ==

A plugin which Create a galley, the gallery show in popup slider when click on any single image. 

<a href="http://dewa.org.pk/expo_Comparison/sws-gallery-plugin">See Demo</a>

Complete responsive gallery plugin. Show the images in any coulmns with zooming effect. 

[youtube https://www.youtube.com/watch?v=QcKgO9n5Y2s]


Use the follwing shortcode into page or post.

	[sws_gallery] 

Shortcode Example with attributes

	[sws_gallery column="3" posts_per_page ="8"]

Shortcode Attributes

	1.  column (show the gallery in column, numaric value 1 to 12 deafult value is 3 if not specified)
	2.  posts_per_page (Show number of photos per page numaric value default value is -1 if not specified)


 
Download the plugin and activate it, once you activate it,find SWS Gallery in your admin menu normally its under the pages or media.
click on add new and set the feature image of your post and then publish it.
Insert the shortcode [sws_gallery] in your post or page.

If you want to use it within template use 

	<?php echo do_shortcode('[sws_gallery]');?> 

or 
	
	<?php echo do_shortcode('[sws_gallery column="3" posts_per_page ="8"]');?>

== Installation ==

This section describes how to install the plugin and get it working.


  
   From your WordPress dashboard

    1. Visit 'Plugins > Add New'
    2. Search for 'SWS Gallery'
    3. Install SWS Gallery Plugin
    4. Activate SWS Gallery Gallery from your Plugins page.
    5. Add new SWS Galery set the feature Image and published it.
    6. Add Shortcode into your page or post or in your template.
